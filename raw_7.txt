## 1. Implementation Plan

| # | Area | Action |
|---|------|--------|
| 1 | **Design System** | Adopt a dark casino/showroom palette: deep charcoal/black base, neon amber (`#ffb347`) and electric violet (`#8b5cf6`) accents, glass panels, milky translucent overlays. |
| 2 | **Typography** | Use a clean sans-serif stack (`Inter`, `Segoe UI`, system UI) with generous headings and subtle letter-spacing. |
| 3 | **Glassmorphism** | Apply `rgba(255,255,255,0.05)` backgrounds with `backdrop-filter: blur()` and thin light borders. |
| 4 | **Animations** | Add CSS transitions for hover lifts, pulse rings for âliveâ indicators, and smooth fade-ins for feed parsing. |
| 5 | **Foyer.razor** | Build a branded landing hero with a spotlight effect, tagline, and CTAs to enter the showroom. |
| 6 | **Feed.razor** | Create a three-zone layout: comedian syndicate sidebar (left), main showroom/player (center), live audience chat sidebar (right). |
| 7 | **Responsiveness** | Use CSS Grid with breakpoints so the sidebars collapse on mobile and the player remains dominant. |
| 8 | **Polling & Audio** | Wire `GetChatMessagesAsync()` every 5 seconds; render `<audio controls>` with `preload="none"` against `/api/jokes/{id}/audio`. |
| 9 | **Trigger** | Expose a âCue Comedianâ button wired to `TriggerComedianAsync()`. |
